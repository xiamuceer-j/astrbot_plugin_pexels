# Pexels 精选图片插件

作者：xiamuceer-j

这是一个基于 AstrBot 框架的插件，用于从全球知名的免费图片和视频网站 Pexels.com 获取高质量的精选图片。用户可以通过发送命令快速浏览 Pexels 上的优质摄影作品。

---

## 功能

- **获取图片**: 用户可以通过命令 `/pexel` 随机获取一批来自 Pexels 的精选图片。
- **图片信息**: 返回的结果会包含每张图片的：
    - 摄影师姓名
    - 摄影师 Pexels 主页链接
    - 图片的原图链接
- **查看帮助**: 用户可以通过命令 `/pexelhelp` 查看本插件的指令说明。
